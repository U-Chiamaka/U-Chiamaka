USDA Pricing Data Analysis

In this project, I analyzed agricultural pricing trends from USDA datasets using Power BI. Online Learning Data Analysis

This project explores trends in online learning across Spain, India, and France from 2004 to 2021. Using Python and Plotly, I analyzed growth patterns, identified peak years, and visualized regional differences in adoption rates. The insights highlight the evolution of e-learning globally and provide a foundation for understanding future trends in digital education.

Data Cleaning

This project focused on cleaning and organizing a dataset with multiple errors, inconsistencies, and unwanted characters. Using Python and Pandas, I standardized formats, removed invalid entries, handled missing values, and ensured the data was clean, structured, and ready for analysis.

This project involved analyzing perfume sales data using Python, SQL, and Power BI to uncover key trends and insights. The analysis highlighted sales dynamics and provided actionable recommendations for optimizing inventory and marketing strategies.
